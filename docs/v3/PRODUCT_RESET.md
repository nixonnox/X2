# PRODUCT_RESET — X2 Platform

> 작성일: 2026-03-10
> 상태: 확정

---

## 1. 제품 한 줄 정의

**"소셜 + 검색 + 댓글 데이터를 하나의 인텔리전스 루프로 연결하여, 마케터가 '무슨 일이 일어났고 / 왜 일어났고 / 무엇을 해야 하는가'를 한 곳에서 확인하고 실행까지 이어갈 수 있는 Marketing Intelligence & Execution Platform."**

---

## 2. 핵심 사용자 문제

### 2.1 데이터는 있는데 인사이트가 없다

마케터는 YouTube Analytics, Instagram Insights, Google Search Console, 소셜 리스닝 툴, 키워드 분석 툴을 각각 따로 열어본다. 데이터는 넘치지만 "왜 이런 결과가 나왔고, 다음에 무엇을 해야 하는가?"에 대한 답을 직접 조합해야 한다.

### 2.2 분석에서 끝나고 실행으로 이어지지 않는다

기존 도구들은 차트와 숫자를 보여주지만, 구체적인 액션(콘텐츠 제작, 댓글 대응, 인플루언서 섭외, 캠페인 실행)까지 연결해주지 않는다. 분석 → 실행 사이에 단절이 존재한다.

### 2.3 AI 검색 시대에 인용 구조를 관리하지 못한다

Google AI Overview, Perplexity, ChatGPT Search 등 AI 답변 엔진이 대두되면서, 자사 콘텐츠가 AI에 인용되는지, 경쟁사는 인용되는데 우리는 빠져 있는지를 추적하고 최적화할 도구가 없다.

### 2.4 산업별 맥락이 반영되지 않는 범용 분석

뷰티, F&B, IT/SaaS, 패션 등 산업마다 중요한 지표와 벤치마크가 다르지만, 기존 도구는 범용 대시보드만 제공한다.

## 3. 기존 툴과의 차이

| 기존 도구             | 하는 일                         | 못하는 일                   | X2가 해결하는 방식              |
| --------------------- | ------------------------------- | --------------------------- | ------------------------------- |
| **리스닝마인드**      | 검색 인텐트, 클러스터, 저니     | 소셜/댓글 연결, 실행        | Intent → Social → Action 연결   |
| **썸트렌드**          | 소셜 트렌드 차트, 보고서용 근거 | 검색 인텐트, 캠페인 실행    | 근거형 UX + 인텐트/실행 통합    |
| **소머즈**            | Raw 소셜 멘션, 필터, 다운로드   | AI 분석, 인사이트, 실행     | Raw 데이터 + AI 분석 + 액션     |
| **피처링**            | 인플루언서 탐색, 캠페인 실행    | 검색 인텐트, 댓글 심층 분석 | 분석→실행→측정 풀루프           |
| **아하트렌드**        | 산업별 벤치마크                 | 소셜/검색 통합, 실행        | Vertical Pack + 통합 인텔리전스 |
| **YouTube Analytics** | 자사 채널 데이터                | 경쟁 분석, 검색 연결, 액션  | 자사+경쟁+검색+실행 통합        |

### 핵심 차별화

1. **멀티 엔트리, 싱글 인텔리전스**: 소셜/리스닝/댓글 어디서 시작해도 하나의 인텔리전스 흐름으로 수렴
2. **분석 → 실행 → 측정 → 재분석 루프**: 인사이트가 액션이 되고, 액션의 결과가 다시 분석으로 돌아옴
3. **GEO/AEO 네이티브**: AI 검색 인용 추적과 최적화가 플랫폼에 내장
4. **산업별 맥락 (Vertical Pack)**: 뷰티, F&B 등 산업별 벤치마크와 템플릿 내장

## 4. 플랫폼의 최종 가치 제안

```
Input:  채널 URL / 키워드 / 댓글 데이터
Output: "무슨 일이 일어났고 / 왜 일어났고 / 무엇을 해야 하는가"
        + 실행 + 실행 결과 측정 + 재분석
```

### 사용자에게 전달하는 가치

| 가치              | 설명                                                  |
| ----------------- | ----------------------------------------------------- |
| **시간 단축**     | 5개 도구를 따로 쓰던 것을 1개로                       |
| **인사이트 품질** | 소셜+검색+댓글 교차 분석으로 단일 소스 대비 깊은 해석 |
| **실행 전환율**   | 분석에서 끝나지 않고 액션까지 연결                    |
| **AI 검색 대응**  | GEO/AEO 추적으로 AI 시대 SEO 선점                     |
| **보고 효율**     | 근거 기반 리포트 자동 생성, 공유, 발송                |
| **산업 맥락**     | 범용이 아닌 산업별 벤치마크와 인사이트                |

## 5. 왜 이 서비스가 필요한지

### 5.1 시장 변화

- **AI 검색 대두**: Google AI Overview가 검색 결과의 40%+ 영역을 차지. 기존 SEO만으로는 불충분.
- **소셜 플랫폼 다양화**: YouTube, Instagram, TikTok, X — 채널이 늘어날수록 통합 분석의 필요성 증가.
- **마케팅 효율 압박**: 예산 대비 성과를 증명해야 하는 환경. "감"이 아닌 "근거" 기반 의사결정 필수.

### 5.2 기존 시장의 공백

```
[리스닝마인드] ──── 검색 인텐트 ─────┐
[썸트렌드]   ──── 소셜 트렌드 ──────┤
[소머즈]    ──── Raw 멘션 ──────────┤──→ 여기가 비어 있다
[피처링]    ──── 인플루언서/캠페인 ──┤    (통합 + 실행 + 측정 + AI 인용)
[아하트렌드] ──── 산업 벤치마크 ─────┘
```

각각은 자기 영역에서 강하지만, 이를 하나의 인텔리전스 루프로 연결하는 플랫폼은 없다. X2는 이 공백을 채운다.

### 5.3 기술적 타이밍

- LLM 기반 분석의 성능이 상용 수준에 도달
- 각 소셜 플랫폼의 공식 API가 성숙
- AI 검색 엔진의 답변 구조가 표준화되기 시작
- 이 세 가지가 동시에 가능해진 시점이 지금

## 6. 7-Stage Intelligence Loop

```
① Discover → ② Analyze → ③ Intent → ④ GEO/AEO
                                          ↓
⑦ Measure ← ⑥ Execute ← ⑤ Insight
    └──────────→ ① Discover (재순환)
```

| Stage      | 핵심 활동                                  | 주요 데이터                                             |
| ---------- | ------------------------------------------ | ------------------------------------------------------- |
| ① Discover | 채널/키워드/트렌드/경쟁 신호/Raw 멘션 수집 | Channel, Content, RawSocialMention, Keyword             |
| ② Analyze  | 채널/콘텐츠/댓글/경쟁 분석                 | ChannelSnapshot, ContentMetric, CommentAnalysis         |
| ③ Intent   | 검색 의도 분류, 클러스터, 저니, FAQ 해석   | IntentQuery, IntentKeywordResult, TrendKeywordAnalytics |
| ④ GEO/AEO  | AI 인용 추적, 출처 최적화                  | AeoKeyword, AeoSnapshot, CitationReadyReportSource      |
| ⑤ Insight  | 전략 인사이트, 기회/리스크, 경쟁 갭        | InsightReport, ReportSection, EvidenceAsset             |
| ⑥ Execute  | 콘텐츠/운영/인플루언서/캠페인 액션         | InsightAction, Campaign, CampaignCreator                |
| ⑦ Measure  | 성과 측정, ROI 산출, 재실행 판단           | CampaignMetric, PostMeasurement, RoiCalculation         |

---

## 7. 불변 원칙

1. **Mock/Hardcoded 데이터 금지** — Fallback 및 테스트 전용만 허용
2. **공식 API 우선** — 웹 수집 시 robots.txt, 서비스 약관, 공개 접근 정책 준수
3. **비인가 접근/인증 우회/차단 회피 설계 금지**
4. **GEO/AEO Citation 추적 필수** — AI 검색 인용 트래킹 + 출처 우선순위 관리
5. **실행-측정 루프 필수** — 분석만으로 끝나지 않고 Execute → Measure → Re-Discover 순환
6. **산업별 맥락 (Vertical Pack)** — 범용 분석이 아닌 산업별 벤치마크/템플릿 지원
