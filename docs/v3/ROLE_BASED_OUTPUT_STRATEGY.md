# 역할별 출력 전략

## 4가지 역할 정의

| 역할               | 코드 상수      | 주요 관심사              | 출력 특성                        |
| ------------------ | -------------- | ------------------------ | -------------------------------- |
| 실무자/일반 직장인 | `PRACTITIONER` | 근거 자료, 차트, 원문    | PPT 친화형, 상세 데이터 포함     |
| 마케터/AE          | `MARKETER`     | 실행 액션, 채널 포인트   | 액션 중심, 다음 콘텐츠 제안 포함 |
| 관리자/운영자      | `ADMIN`        | 리스크, 상태, 누락, 지연 | 운영 현황 + 이상 징후 중심       |
| 대표/CMO           | `EXECUTIVE`    | 핵심 변화, 전략 요약     | 간결한 전략적 시사점             |

---

## 서비스별 역할 적용

### 1. InsightSummaryService

SummaryContext에 따라 출력 형식을 조절한다:

| SummaryContext | 출력 형식             | 용도                    |
| -------------- | --------------------- | ----------------------- |
| `DASHBOARD`    | 짧은 카드형 요약      | 대시보드 위젯           |
| `EMAIL`        | 이메일 본문 적합 포맷 | 주간/월간 이메일 리포트 |
| `REPORT_INTRO` | 리포트 서두 요약문    | 리포트 첫 페이지        |
| `SLACK`        | 간결한 알림 메시지    | Slack 채널 알림         |

### 2. ExecutiveSummaryService

`EXECUTIVE` 역할 전용 서비스로, 다음 요소를 포함한다:

- **KPI 요약**: 핵심 지표의 변화량과 방향
- **전략적 권고**: 의사결정에 필요한 1-3개 핵심 제안
- **성장 지표**: 채널/캠페인 성장 추이 요약

### 3. ReportCompositionService

`roleContext`에 따라 리포트 섹션 구성이 달라진다:

| roleContext    | 포함 섹션                             | 특징                                 |
| -------------- | ------------------------------------- | ------------------------------------ |
| `EXECUTIVE`    | 전략 개요 + KPI + 권고만              | 1-2페이지 분량, 핵심만               |
| `PRACTITIONER` | 전체 섹션 + 상세 증거                 | 10+ 페이지, PPT용 차트/테이블 포함   |
| `MARKETER`     | 액션 + 콘텐츠 제안 강조               | 실행 로드맵, 채널별 제안 포함        |
| `ADMIN`        | 리스크 + 운영 상태 + 재시도 필요 영역 | 이상 징후 하이라이트, 조치 필요 항목 |

## 같은 데이터, 다른 표현 예시

동일한 분석 결과(부정 댓글 42%)가 역할에 따라 어떻게 다르게 표현되는지 보여준다:

### 원본 데이터

> 전체 댓글 458건 중 부정 댓글 192건 (42%). 주요 토픽: 배송, 품질.

---

### PRACTITIONER (실무자)

> "부정 댓글 192건/458건 (42%). 주요 토픽: 배송, 품질. [상세 표 첨부]"

- 정확한 수치와 비율 제공
- 토픽별 분류 테이블 첨부
- 대표 부정 댓글 샘플 인용
- PPT 슬라이드용 차트 데이터 포함

### MARKETER (마케터)

> "부정 반응 급증. 배송 관련 FAQ 콘텐츠 제작 권고. 예상 효과: 부정 비율 10%p 감소"

- 현상보다 **다음 행동**에 초점
- 구체적 콘텐츠 제안 포함
- 예상 효과/ROI 언급
- 채널별 대응 포인트 제시

### ADMIN (관리자)

> "⚠ 부정 비율 42% — 전월 대비 +15%p. 리스크 시그널 3건 활성. 대응 필요"

- 이상 징후 강조 (⚠ 아이콘)
- 전월 대비 변화량 명시
- 활성 리스크 시그널 수 표시
- 대응 필요 여부 즉시 판단 가능

### EXECUTIVE (대표/CMO)

> "고객 불만 심화 추세. 배송 프로세스 개선 투자 검토 권고"

- 한 문장으로 핵심 전달
- 전략적 의사결정 포인트 제시
- 세부 수치 생략, 방향성만 제공
- 투자/리소스 배분 관점의 권고
