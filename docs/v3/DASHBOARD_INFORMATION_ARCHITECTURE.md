# Dashboard Information Architecture (Phase 8)

## 설계 원칙

1. **인사이트 중심**: "오늘 무엇을 봐야 하고, 무엇을 해야 하는지"가 첫 화면에 보인다
2. **빈 상태 ≠ Mock 데이터**: 데이터가 없으면 구조 미리보기와 시작 안내를 보여준다
3. **역할별 분기 준비**: PRACTITIONER/MARKETER/ADMIN/EXECUTIVE 역할에 따라 다른 위젯 구성 가능
4. **근거 추적 가능**: 모든 인사이트는 Evidence Bundle을 통해 원본 데이터까지 추적 가능

## 인사이트 카드 구조

각 인사이트 카드는 다음 정보를 표시한다:

| 필드     | 출처                          | 표시 방식                                        |
| -------- | ----------------------------- | ------------------------------------------------ |
| 카테고리 | `insight.category`            | 뱃지 (KEY_FINDING/OPPORTUNITY/RISK/TREND_CHANGE) |
| 제목     | `insight.title`               | 카드 헤더                                        |
| 요약     | `insight.narrative`           | 한국어 1-2문장                                   |
| 심각도   | `insight.severity`            | 색상 코딩 (high=red, medium=yellow, low=green)   |
| 근거 수  | `insight.evidenceRefs.length` | "근거 N건" 링크 → Evidence 탭                    |

---

## 액션 카드 구조

| 필드     | 출처              | 표시 방식                                   |
| -------- | ----------------- | ------------------------------------------- |
| 제목     | `action.title`    | 행 제목                                     |
| 카테고리 | `action.category` | 뱃지 (CONTENT_CREATION/SEO_OPTIMIZATION 등) |
| 우선순위 | `action.priority` | HIGH/MEDIUM/LOW 뱃지                        |
| 담당자   | `action.owner`    | 한국어 역할명 (콘텐츠 팀/SEO 팀 등)         |
| 기한     | `action.timing`   | 즉시/1주 내/1개월 내                        |
| 근거     | `action.evidence` | "관련 인사이트 보기" 링크                   |

## 빈 상태 전략

| 상태          | 표시 내용                                                                               |
| ------------- | --------------------------------------------------------------------------------------- |
| 채널 없음     | "채널을 등록하면 분석이 시작됩니다" + `/channels/new` CTA                               |
| 수집 중       | "데이터를 수집하고 있습니다. 잠시 후 인사이트가 표시됩니다"                             |
| 분석 전       | "수집된 데이터를 분석하고 있습니다" + 진행률                                            |
| 인사이트 없음 | 카드 구조 미리보기 (회색 플레이스홀더) + "분석이 완료되면 인사이트가 여기에 표시됩니다" |
| 에러          | "데이터를 불러오는 중 오류가 발생했습니다" + 재시도 버튼                                |
