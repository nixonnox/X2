# Intelligence Drill-Down Architecture

## 개요

방사형 그래프 노드 클릭 → 상세 패널 구조

## 노드 유형별 drill-down

- **seed**: 전체 요약 + 주요 블록 수 + 신호 상태
- **taxonomy**: 카테고리명 + 매핑된 클러스터 수 + 관련 블록
- **block**: 블록 제목 + taxonomy 태그 + 관련 문장

## EvidenceSidePanel 구조

- **노드 메타**: type, label, clusterCount, taxonomyTags
- **요약**: 노드 유형별 한국어 요약 문장
- **관련 키워드/질문**: 블록 문장에서 추출, 최대 5개
- **관련 데이터 블록**: taxonomy/type 기반 필터링
- **벤치마크 맥락**: baseline에서 관련 메트릭 표시
- **소셜 신호**: 멘션 수, 감성, 플랫폼
- **커버리지 상태**: confidence bar, 벤치마크 유무, 소셜 유무

## 반응형 동작

- **데스크톱**: 우측 slide-in 패널 (400px)
- **모바일**: 하단 bottom sheet (최대 80vh, 둥근 상단 모서리)

## 상태 처리

- **관련 데이터 없음**: "관련 데이터 없음" 회색 텍스트
- **벤치마크 없음**: "이 노드의 벤치마크 비교 데이터가 없습니다"
- **소셜 없음**: "이 노드와 관련된 소셜 신호가 없습니다"
- **저신뢰**: "신뢰도가 낮은 노드입니다. 참고 수준으로 활용하세요."
