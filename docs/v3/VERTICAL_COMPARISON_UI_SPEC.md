## 2. 컴포넌트 매핑

| 영역 | 컴포넌트 | ViewModel |
|------|---------|-----------|
| 상단 카드 3개 | VerticalPreviewBoard (내장) | inputSummary, suggestionBadge, overallScore |
| 업종 헤더 | VerticalPreviewBoard (내장) | industryColumns |
| 비교 매트릭스 | VerticalComparisonMatrix | comparisonSections |
| 차이점 요약 | VerticalDifferencePanel | differenceSummary |
| 상태 패널 | VerticalPreviewStatePanel | screenState |

## 3. 상호작용

### 3.1 아코디언 동작
- 차이가 있는 섹션은 **자동 펼침**
- 차이가 없는 섹션은 **접힌 상태**
- 클릭으로 토글
- 섹션 헤더에 차이 건수 뱃지 표시

### 3.2 셀 강조
| 수준 | 색상 | 의미 |
|------|------|------|
| CRITICAL | 빨간 배경 + 링 | 치명적 차이 (품질 미달 등) |
| WARNING | 노란 배경 | 주의 필요 (톤, 경고, 숨김 블록) |
| INFO | 파란 배경 | 정보성 차이 (이상값) |
| (없음) | 배경 없음 | 차이 없음 |

### 3.3 이상값 표시
- 4개 업종 중 혼자만 다른 값 → 파란 점(●) 표시
- 호버 시 이유 표시 (title 속성)

### 3.4 품질 경고 배너
| 조건 | 표시 |
|------|------|
| Mock 데이터 | 노란 배너: "Mock 데이터 기반" |
| Stale 데이터 | 노란 배너: "Stale 데이터" |
| Partial 데이터 | 노란 배너: "Partial 데이터" |

## 4. 상태별 화면

| 상태 | 표시 |
|------|------|
| idle | 안내 문구 ("키워드와 문서 유형을 선택하고 비교를 시작하세요") |
| loading | 스피너 + "4개 업종 비교 프리뷰 생성 중..." |
| error | 빨간 알림 + 에러 메시지 |
| success + empty | "비교 결과가 없습니다" |
| success + data | 전체 프리뷰 화면 |

## 5. 색상 체계

| 업종 | 메인 색상 | 배경 | 테두리 |
|------|----------|------|--------|
| 뷰티 | pink-600 | pink-50 | pink-200 |
| F&B | orange-600 | orange-50 | orange-200 |
| 금융 | blue-600 | blue-50 | blue-200 |
| 엔터 | purple-600 | purple-50 | purple-200 |

## 6. 반응형

- **1200px+**: 4컬럼 테이블 (전체 표시)
- **768px-1199px**: 가로 스크롤 가능한 테이블
- **~767px**: 카드 레이아웃 (업종별 카드 세로 배치)
