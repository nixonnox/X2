## 2. DNA별 상세 매핑

### 2.1 리스닝마인드 DNA

```
리스닝마인드의 핵심: "사람들이 무엇을 검색하고, 왜 검색하고, 어떤 경로로 검색하는가"

X2에서의 반영:

[Intent Map]
  사용자가 시드 키워드 입력
  → IntentQuery 생성 (status: QUEUED)
  → 키워드 확장 (자동완성/연관검색)
  → LLM 인텐트 분류
  → IntentKeywordResult 저장
  → 결과: 인텐트 분류표 + 갭 스코어

[Cluster Map]
  IntentKeywordResult 목록
  → LLM/임베딩 기반 의미 클러스터링
  → IntentQuery.resultGraph에 노드/엣지 저장
  → 결과: 키워드 관계 그래프 시각화

[Journey Map]
  클러스터 + 인텐트 카테고리
  → LLM 경로 추론 (DISCOVERY → COMPARISON → ACTION)
  → IntentQuery.resultGraph에 journey 저장
  → 결과: 검색 여정 플로우 시각화

차별화: 리스닝마인드는 검색만. X2는 검색 + 소셜 + 댓글 교차.
```

### 2.2 썸트렌드 DNA

```
썸트렌드의 핵심: "직장인이 기획서/보고서에 바로 쓸 수 있는 근거 차트"

X2에서의 반영:

[Evidence Block UX]
  ReportSection 내 EvidenceAsset
  → type: METRIC | CHART | TABLE | QUOTE | COMPARISON | RANKING
  → dataSourceType + dataEntityIds → 실데이터 참조 (Mock 금지)
  → visualization: {chartType, config}
  → isEditable: true → 사용자가 차트 커스텀 가능

[보고서 빌더]
  ReportTemplate → ReportSection 구조 정의
  → AI 자동 생성 (isAutoGenerated) + 수동 편집
  → {{evidence:ID}} 마커로 본문 내 근거 인라인 배치
  → PDF/PPT 내보내기 (향후)

[키워드 추이]
  TrendKeywordAnalytics
  → 검색량 + 소셜 언급량 교차 차트
  → seasonalityScore: 시즌 의존도

차별화: 썸트렌드는 소셜 트렌드 중심. X2는 검색 + 인텐트 + GEO/AEO까지 근거 확장.
```

### 2.3 소머즈 DNA

```
소머즈의 핵심: "원시 소셜 데이터를 내가 직접 보고, 필터링하고, 내려받는다"

X2에서의 반영:

[Data Explorer]
  RawSocialMention 테이블
  → 필터: platform, publishedAt, matchedKeyword, sentiment, mediaType
  → 정렬: 최신순, 참여율순, 좋아요순
  → 원문 전체 열람

[SavedFilter]
  자주 쓰는 필터 조건 저장
  → 팀 공유 (isShared)

[DataExportJob]
  CSV / Excel / JSON 내보내기
  → 비동기 생성 (큐)
  → 다운로드 링크 (7일 유효)

차별화: 소머즈는 멘션만. X2는 멘션 + 댓글 + 콘텐츠 + 키워드 + 인텐트 모두 탐색 가능.
```

### 2.4 아하트렌드 DNA

```
아하트렌드의 핵심: "산업별 맥락이 반영된 분석과 벤치마크"

X2에서의 반영:

[VerticalPack]
  industry: BEAUTY | FOOD_BEVERAGE | FASHION | TECH_SAAS | ...
  seedKeywords: 산업별 핵심 키워드 프리셋
  benchmarkBaseline: 산업 평균 참여율, 성장률 등
  topicTaxonomy: 산업별 토픽 분류 체계
  competitorPresets: 주요 경쟁 채널 프리셋

[WorkspaceVerticalPack]
  Workspace ↔ VerticalPack M:N 연결
  → 활성화 시 해당 산업 벤치마크/템플릿 적용
  → 분석 시 산업 평균 대비 비교 자동 표시

[ReportTemplate]
  verticalPackId → 산업별 리포트 템플릿
  → "뷰티 브랜드 월간 리포트", "F&B 경쟁 분석 리포트" 등

차별화: 아하트렌드는 산업 벤치마크만. X2는 벤치마크 + 실행 + 측정까지 산업 맥락 적용.
```

### 2.5 디센트릭 DNA

```
디센트릭의 핵심: "엔터프라이즈 고객을 위한 커스텀 분석/대시보드/API"

X2에서의 반영:

[Enterprise Tier]
  Workspace.planTier = ENTERPRISE
  → SSO (SAML/OIDC) 연동 (향후)
  → SLA 보장
  → 화이트라벨 (로고/도메인 커스텀)
  → 전용 API 접근

[커스텀 분석]
  ReportTemplate (isSystem: false)
  → 사용자 정의 섹션/데이터 조합
  → 커스텀 대시보드 (향후 대시보드 빌더)

[API 통합]
  Public REST/GraphQL API (향후)
  → 외부 시스템 데이터 연동
  → Webhook 이벤트 전달

상태: Enterprise 기능은 Phase 8. 현재는 구조만 설계.
```

### 2.6 피처링 DNA

```
피처링의 핵심: "인플루언서 검색 → 캠페인 실행 → 성과 추적 → 재분석 루프"

X2에서의 반영:

[Influencer Discovery]
  Channel (channelType: INFLUENCER)
  → InfluencerProfile 1:1 확장
  → tierLevel: NANO ~ MEGA
  → overallScore: 0~100 종합 점수
  → scoreBreakdown: {reach, engagement, consistency, growth}

[Campaign Execution]
  Campaign → CampaignCreator → CampaignContent
  → outreachStatus: PROPOSED → NEGOTIATING → CONTRACTED → CREATING → PUBLISHED
  → compensationType, deliverables, contractUrl

[Performance Tracking]
  CampaignContent → PostMeasurement (일별)
  Campaign → CampaignMetric (일별 집계)
  → RoiCalculation: ROI, ROAS, CPM, CPV, CPE

[Loop]
  Measure 결과 → InsightAction 생성 → 재분석/재실행
  RoiCalculation.aiSummary → "이 캠페인의 CPV가 산업 평균 대비 30% 낮음. 마이크로 인플루언서 비중 확대 권고."

차별화: 피처링은 인플루언서/캠페인만. X2는 캠페인 성과를 검색 인텐트/GEO/AEO와 교차 분석.
```
