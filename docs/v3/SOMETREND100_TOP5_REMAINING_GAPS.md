# Sometrend 100%: Top 5 Remaining Gaps

> Date: 2026-03-16

## 썸트렌드 대체에 가장 큰 영향을 주는 미완료 5개

| # | Gap | 영향도 | 분류 | 난이도 |
|---|-----|--------|------|--------|
| **1** | **네이버 카페/커뮤니티 데이터** | 매우 높음 | 법적 보류 | VERY HIGH — 합법적 경로 없음 |
| **2** | **차트 이미지 다운로드** | 높음 | S3 | MEDIUM — html2canvas/SVG export |
| **3** | **API key 관리 + Rate limit** | 높음 | S2 | MEDIUM — Redis + UI |
| **4** | **Stripe 결제 연동** | 중간 | S2 | MEDIUM — Stripe SDK |
| **5** | **업종별 속성/연관어 차별화** | 중간 | S3 | LOW — ATTRIBUTES 매핑 확장 |

## 영향도 분석

### #1. 네이버 카페/커뮤니티
- 썸트렌드의 핵심 차별점 — 국내 커뮤니티 데이터
- x2는 YouTube+네이버 블로그/뉴스까지만
- **법적 제약으로 직접 해결 불가** — 제3자 데이터 제공업체 또는 공식 파트너십 필요

### #2. 차트 이미지 다운로드
- 보고서 활용에 직접 영향
- CSV는 있지만 차트 캡처가 없으면 "썸트렌드처럼 캡처해서 붙이기" 불가

### #3. API key + Rate limit
- 외부 연동 상품으로 제공하려면 필수
- 현재 간단 Bearer 인증만

### #4. Stripe
- 유료 전환에 필수
- 필드는 준비되어 있음

### #5. 업종별 차별화
- "뷰티는 성분/텍스처 속성으로, 금융은 금리/수수료 속성으로" 구분
- 현재 5개 속성이 모든 업종에 동일
