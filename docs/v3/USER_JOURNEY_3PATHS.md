# USER_JOURNEY_3PATHS — 3가지 시작점, 하나의 도착점

> 작성일: 2026-03-10
> 상태: 확정

---

## 원칙

**멀티 엔트리, 싱글 인텔리전스 플로우**

- 시작은 여러 개 (소셜, 리스닝, 댓글)
- 도착점은 하나: **"무슨 일이 일어났고 / 왜 일어났고 / 무엇을 해야 하는가"**
- 도착점에서 실행(Execute)으로 이어지고, 실행 결과가 다시 시작점으로 되돌아온다.

---

## 공통 도착점 (Convergence Point)

모든 경로는 아래 상태로 수렴한다:

```
┌─────────────────────────────────────────────┐
│ Intelligence Dashboard                       │
│                                              │
│ ① 무슨 일이 일어났는가 (Facts)              │
│    - 채널/콘텐츠/댓글 성과 요약              │
│    - 트렌드/키워드 변동                      │
│    - 경쟁 채널 비교                          │
│                                              │
│ ② 왜 일어났는가 (Why)                       │
│    - 인텐트/클러스터 분석                    │
│    - 감성/토픽/FAQ 해석                      │
│    - 경쟁 갭 / 기회 식별                     │
│    - GEO/AEO 인용 변동 원인                  │
│                                              │
│ ③ 무엇을 해야 하는가 (Actions)              │
│    - 우선순위 액션 리스트                    │
│    - 콘텐츠/운영/인플루언서/캠페인 제안      │
│    - 리포트 생성 / 공유 / 발송               │
│    - 실행 → 측정 → 재분석 루프              │
└─────────────────────────────────────────────┘
```

## Path 1: 소셜 시작형

> "내 채널이 잘 되고 있는지, 경쟁사 대비 어떤지 알고 싶다"

### 대상 사용자

- 브랜드 소셜 미디어 매니저
- 콘텐츠 크리에이터
- 마케팅 팀 리드

### 단계별 행동 흐름

```
[1] 채널 등록
    └→ YouTube/Instagram/TikTok/X 채널 URL 입력 또는 API 연결
    └→ 채널 기본 정보 수집 (구독자, 콘텐츠 수, 프로필)
    └→ 자사 채널 / 경쟁 채널 구분 설정

[2] 콘텐츠 & 댓글 수집
    └→ 공식 API로 최근 콘텐츠 목록 수집
    └→ 콘텐츠별 조회수/좋아요/댓글수/참여율 수집
    └→ 댓글 수집 및 감성/토픽/FAQ/리스크 분석

[3] 채널 분석 (Analyze)
    └→ 일별 채널 스냅샷 (구독자 추이, 콘텐츠 퍼포먼스)
    └→ 콘텐츠 유형별 성과 비교
    └→ 참여율 트렌드, 성장률

[4] 경쟁 채널 비교 (Analyze)
    └→ 경쟁 채널의 동일 지표 수집
    └→ 벤치마크 비교 (구독자, 참여율, 콘텐츠 빈도)
    └→ 경쟁 갭 식별

[5] 리스닝/키워드 확장 (Discover → Intent)
    └→ 콘텐츠 키워드 자동 추출
    └→ 관련 검색 인텐트 분석
    └→ 키워드 트렌드 + 소셜 멘션 교차 분석
    └→ FAQ / 이슈 해석

[6] GEO/AEO 연결 (GEO/AEO)
    └→ 자사 콘텐츠의 AI 검색 인용 여부 확인
    └→ 경쟁사 인용 현황 대비

[7] 인사이트 (Insight)
    └→ "이 채널은 참여율은 높지만 검색 유입이 약하다"
    └→ "경쟁사 대비 댓글 감성이 부정적이다"
    └→ "이 키워드에서 콘텐츠 공급이 부족하다"

[8] 액션 & 실행 (Execute)
    └→ 콘텐츠 제작 제안
    └→ 댓글 대응 가이드
    └→ 인플루언서 협업 제안
    └→ 캠페인 생성 → 실행

[9] 리포트 & 측정 (Measure)
    └→ 주간/월간 리포트 자동 생성
    └→ 공유 링크, 이메일 발송
    └→ 캠페인 성과 재측정 → 재분석 루프
```

### 주요 화면 흐름

```
채널 등록 → 채널 대시보드 → 콘텐츠 목록 → 댓글 분석
  → 경쟁 비교 → 키워드/인텐트 → GEO/AEO
  → 인사이트 허브 → 액션 보드 → 리포트 → 캠페인
```

## Path 2: 리스닝 시작형

> "우리 브랜드/제품이 어떻게 언급되고, 사람들이 무엇을 검색하는지 알고 싶다"

### 대상 사용자

- PR / 커뮤니케이션 매니저
- 브랜드 전략 담당자
- 시장 조사 담당자

### 단계별 행동 흐름

```
[1] 키워드 등록
    └→ 브랜드명, 제품명, 경쟁사명, 관심 키워드 입력
    └→ 모니터링 대상 플랫폼 선택

[2] 소셜 멘션 수집 (Discover)
    └→ 등록 키워드로 Raw 소셜 멘션 수집
    └→ 플랫폼별 언급량, 감성, 토픽 분석
    └→ Data Explorer (소머즈형): 원문 열람, 필터, 다운로드

[3] 검색 인텐트 분석 (Intent)
    └→ 키워드 인텐트 분류 (탐색/비교/행동/문제해결)
    └→ 클러스터 맵: 관련 키워드 그룹핑
    └→ 저니 분석: 사용자 검색 경로 추적
    └→ FAQ / 이슈 자동 추출

[4] 트렌드 분석 (Analyze)
    └→ 키워드 검색량 추이
    └→ 소셜 언급량 추이
    └→ 검색 vs 소셜 갭 스코어
    └→ 시즈널리티 분석

[5] 관련 채널/콘텐츠 연결 (Discover → Analyze)
    └→ 해당 키워드로 성과 높은 채널/콘텐츠 자동 발견
    └→ 인플루언서 탐색 (피처링형)
    └→ 경쟁 채널/콘텐츠 매핑

[6] 댓글 결합 (Analyze → Intent)
    └→ 관련 콘텐츠의 댓글 수집
    └→ 감성/토픽/FAQ 교차 분석
    └→ "사람들이 검색하는 것과 댓글에서 말하는 것의 차이"

[7] GEO/AEO 연결 (GEO/AEO)
    └→ 등록 키워드의 AI 검색 답변 추적
    └→ 자사 콘텐츠 인용 여부, 순위
    └→ 인용 대응 소스 등록 및 최적화

[8] 인사이트 (Insight)
    └→ "이 키워드는 검색량은 높지만 소셜 콘텐츠가 부족하다 (Blue Ocean)"
    └→ "경쟁사는 AI 검색에 인용되는데 우리는 빠져있다"
    └→ "댓글에서 반복되는 불만이 있다"

[9] 액션 & 실행 (Execute)
    └→ 키워드 타겟 콘텐츠 제작 제안
    └→ GEO 최적화 액션
    └→ 인플루언서 섭외 / 캠페인 실행

[10] 리포트 & 측정 (Measure)
     └→ 인텐트 분석 리포트
     └→ 멘션 동향 리포트
     └→ 캠페인 성과 → 키워드 재분석 루프
```

### 주요 화면 흐름

```
키워드 등록 → 멘션 대시보드/Data Explorer → 인텐트 맵
  → 트렌드 분석 → 관련 채널/콘텐츠 → 댓글 분석
  → GEO/AEO → 인사이트 허브 → 액션 보드 → 리포트 → 캠페인
```

## Path 3: 댓글 시작형

> "고객이 어떤 반응을 보이는지, 이슈/불만/질문이 무엇인지 알고 싶다"

### 대상 사용자

- CS / CX 매니저
- 제품 매니저
- 커뮤니티 매니저

### 단계별 행동 흐름

```
[1] 댓글 수집 대상 설정
    └→ 자사 채널의 모든 콘텐츠 댓글 수집
    └→ 또는 특정 콘텐츠/기간 지정
    └→ 경쟁 채널 댓글도 수집 가능

[2] 감성/토픽/FAQ 분석 (Analyze → Intent)
    └→ 감성 분석: 긍정/중립/부정 + 판단 근거
    └→ 토픽 분류: 자동 카테고리 분류
    └→ FAQ 추출: 반복 질문 자동 식별
    └→ 리스크 감지: 부정 이슈, 불만, 위험 신호

[3] 이슈/반응 이유 해석 (Intent)
    └→ "왜 부정 댓글이 급증했는가?"
    └→ 시간별/콘텐츠별 감성 변화 추적
    └→ 댓글 토픽과 검색 인텐트 교차

[4] 관련 채널/콘텐츠 연결 (Discover → Analyze)
    └→ 이슈 관련 콘텐츠 자동 식별
    └→ 동일 주제 경쟁 채널 댓글 비교
    └→ 관련 키워드/검색 트렌드 매핑

[5] 검색 의도 연결 (Intent)
    └→ 댓글 FAQ → 관련 검색 인텐트 매핑
    └→ "사람들이 댓글로 묻는 것 = 검색으로도 찾는 것"
    └→ 콘텐츠 갭 식별

[6] 경쟁/트렌드 결합 (Analyze)
    └→ 경쟁 채널의 동일 이슈 댓글 비교
    └→ 산업 전체 감성 트렌드
    └→ 트렌드 키워드와 댓글 토픽 교차

[7] GEO/AEO 연결 (GEO/AEO)
    └→ FAQ 기반 AI 답변 최적화 콘텐츠 전략
    └→ 자주 묻는 질문 → AI 검색 답변 대응

[8] 인사이트 (Insight)
    └→ "최근 1주간 '배송' 관련 부정 댓글 300% 증가"
    └→ "FAQ 1위 '반품 절차' — 현재 자사 콘텐츠 없음"
    └→ "경쟁사는 같은 이슈에 빠르게 대응 중"

[9] 액션 & 실행 (Execute)
    └→ 댓글 대응 가이드 자동 생성
    └→ FAQ 콘텐츠 제작 제안
    └→ 위험 이슈 에스컬레이션
    └→ 캠페인 생성 (이슈 대응형)

[10] 리포트 & 측정 (Measure)
     └→ 댓글 분석 리포트
     └→ 이슈 대응 결과 추적
     └→ 감성 변화 재측정 → 재분석 루프
```

### 주요 화면 흐름

```
채널/콘텐츠 선택 → 댓글 대시보드 → 감성/토픽/FAQ
  → 이슈 타임라인 → 관련 검색 인텐트 → 경쟁 비교
  → GEO/AEO → 인사이트 허브 → 액션 보드 → 리포트 → 캠페인
```

## 화면 IA (Information Architecture)

### 상단 네비게이션

```
Discover | Analyze | Intent | GEO/AEO | Insight | Execute | Measure
```

### 좌측 사이드바

```
[Project Selector]

Discover
  ├── Channels          (채널 관리)
  ├── Keywords          (키워드 관리)
  ├── Mentions          (소셜 멘션)
  └── Data Explorer     (원시 데이터 탐색)

Analyze
  ├── Channel Analytics (채널 분석)
  ├── Content Analytics (콘텐츠 분석)
  ├── Comment Analytics (댓글 분석)
  └── Competitor        (경쟁 분석)

Intent
  ├── Intent Map        (인텐트 분류)
  ├── Clusters          (클러스터)
  ├── Journey           (검색 저니)
  └── FAQ / Issues      (FAQ/이슈)

GEO/AEO
  ├── Citation Tracker  (인용 추적)
  ├── Source Registry   (소스 관리)
  └── Optimization      (최적화)

Insight
  ├── Dashboard         (인사이트 허브)
  ├── Reports           (리포트 빌더)
  └── Templates         (템플릿)

Execute
  ├── Action Board      (액션 보드)
  ├── Campaigns         (캠페인)
  └── Influencers       (인플루언서)

Measure
  ├── Performance       (성과 측정)
  ├── ROI               (ROI 분석)
  └── Benchmarks        (벤치마크)

Settings
  ├── Workspace
  ├── Members
  ├── Integrations
  └── Billing
```
