# Webhook Test Result Message Guide

> Date: 2026-03-16

## 사용자 피드백 문구

### 성공
> 연결에 성공했어요! 이 URL로 알림을 보낼 수 있어요.

### 인증 오류 (401/403)
> 인증 문제가 있어요. URL이나 토큰을 확인해 주세요.

### 주소 없음 (404)
> 이 주소를 찾을 수 없어요. URL을 다시 확인해 주세요.

### 서버 오류 (5xx)
> 상대 서버에 문제가 있어요. 잠시 후 다시 시도해 주세요.

### 타임아웃
> 응답이 10초 안에 오지 않았어요. URL이나 서버 상태를 확인해 주세요.

### DNS 실패
> 이 주소를 찾을 수 없어요. URL을 다시 확인해 주세요.

### 연결 거부
> 서버가 연결을 거부했어요. 서버가 실행 중인지 확인해 주세요.

### 기타 네트워크 오류
> 연결에 실패했어요: {에러 메시지 앞 100자}

### 저장 시 자동 테스트 warning
> Webhook URL에 연결하지 못했어요. 설정은 저장했지만 URL을 확인해 주세요.
> Webhook URL 테스트에서 HTTP {status} 응답을 받았어요. 설정을 저장했지만 확인해 주세요.
