### X2 비교 방식 카드 문구

| 값 | 제목 | 설명 |
|----|------|------|
| keyword_vs_keyword | 키워드 비교 | 다른 키워드와 인사이트를 비교해요 |
| industry_vs_industry | 업종 비교 | 같은 키워드를 다른 업종에서 봐요 |
| period_vs_period | 기간 비교 | 시간에 따라 어떻게 달라졌는지 봐요 |
