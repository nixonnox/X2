# Card Selection & Decision UI Guide (X2 Project)

> 전역 가이드: ~/.claude/decision-ui-global-guide.md
> 이 파일은 X2 프로젝트에서 카드 선택으로 전환한/할 영역을 관리합니다.

## 전환 완료

### 비교 방식 선택 (Compare 화면)
- **이전:** 작은 칩 버튼 3개 (키워드 비교 / 업종 비교 / 기간 비교)
- **이후:** 카드형 선택 (아이콘 + 제목 + 설명)
- **상태:** DONE

## 전환 대상 (순차 반영)

### 업종 선택 (Intelligence 메인)
- **현재:** 작은 칩 버튼 4개 (뷰티/F&B/금융/엔터)
- **권장:** 카드형 (아이콘 + 업종명 + 설명)
- **우선순위:** HIGH — 첫 화면이므로 가장 먼저

### 알림 채널 선택 (Settings > Notifications)
- **현재:** 체크박스
- **권장:** 카드형 (아이콘 + 채널명 + 상태)
- **우선순위:** MEDIUM

### 문서 형식 선택 (Vertical Preview)
- **현재:** 드롭다운 또는 라디오
- **권장:** 카드형 (아이콘 + 형식명 + 설명)
- **우선순위:** MEDIUM

### 기간 필터 (Notifications 페이지)
- **현재:** 칩 버튼 (전체/오늘/7일/30일)
- **권장:** 칩 유지 — 단순 값이므로 카드 불필요
- **우선순위:** 유지

## 전환 불필요

| 영역 | 이유 |
|------|------|
| 기간 필터 (1일/7일/30일) | 단순 값, 설명 불필요 |
| 읽음/안읽음 필터 | 2개 선택지, 설명 불필요 |
| 우선순위 필터 | 라벨만으로 충분 |
