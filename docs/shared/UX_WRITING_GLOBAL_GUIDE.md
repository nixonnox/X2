## X2 핵심 화면별 문구 정리

### Intelligence 메인 화면

| 위치 | 현재 | 개선 |
|------|------|------|
| 헤더 부제 | 키워드 인텔리전스 · 실시간 신호 · 비교 인사이트 | 키워드를 분석하고, 실시간 반응을 보고, 비교해요 |
| 분석 버튼 | 분석 | 분석 시작 |
| 빈 입력 placeholder | 분석할 키워드를 입력하세요 | 어떤 키워드를 분석해볼까요? |
| 빈 화면 제목 | 키워드를 입력하여 분석을 시작하세요 | 키워드를 입력하면 분석이 시작돼요 |
| 빈 화면 설명 | 검색 클러스터 · 소셜 신호 · 업종 벤치마크를 통합 분석합니다 | 검색 데이터, 소셜 반응, 업종 기준을 함께 분석해요 |
| 시그널 풍부 | 풍부 | 풍부해요 |
| 시그널 보통 | 보통 | 보통이에요 |
| 시그널 최소 | 최소 | 최소한이에요 |
| 로딩 | 인텔리전스 분석 중... | 분석하고 있어요 |
| 과거 뱃지 | 과거 | 이전 데이터 |
| 부분 소셜 | 부분 소셜 | 일부 데이터 |
| 실시간 | 실시간 | 실시간 |

### Compare 화면

| 위치 | 현재 | 개선 |
|------|------|------|
| 제목 | Intelligence A/B 비교 | 비교 분석 |
| 부제 | 두 키워드/업종/기간의 인텔리전스를 비교하여 차이를 시각적으로 확인합니다 | 키워드, 업종, 기간별로 어떻게 다른지 비교해요 |
| 비교 버튼 | 비교 분석 시작 | 비교 시작 |
| 로딩 | A/B 비교 분석 중... | 비교하고 있어요 |
| 핵심 차이점 | 핵심 차이점 | 주요 차이 |
| 변화 하이라이트 | 변화 하이라이트 | 눈에 띄는 변화 |
| 행동 변화 | 행동 변화 | 달라진 점 |

### Bell / 알림

| 위치 | 현재 | 개선 |
|------|------|------|
| 제목 | 알림 | 알림 |
| 전체 읽음 | 모두 읽음 | 모두 읽음 |
| 빈 상태 | 새로운 알림이 없습니다 | 새로운 알림이 없어요 |
| 에러 | 알림 목록을 가져오지 못했습니다 | 알림을 불러오지 못했어요 |
| 로딩 | 알림을 불러오는 중입니다 | 알림을 불러오는 중이에요 |
| 하단 링크 | 모든 알림 보기 | 알림 전체 보기 |

### Current vs Previous

| 위치 | 현재 | 개선 |
|------|------|------|
| 제목 | 현재 vs 이전 | 이전과 비교 |
| no history | 이전 분석 기록이 없습니다 | 비교할 이전 결과가 아직 없어요 |
| no history 설명 | 같은 키워드를 두 번 이상 분석하면 변화를 비교할 수 있습니다 | 같은 키워드를 한 번 더 분석하면 바로 비교할 수 있어요 |
| insufficient | 이전 분석 데이터를 로드할 수 없습니다 | 이전 데이터를 불러오지 못했어요 |
| CTA | 상세 비교 보기 | 자세히 비교하기 |
