# Empty State & Error Copy Guide (X2 Project)

> 전역 라이브러리: ~/.claude/empty-state-global-library.md
> 이 파일은 X2 프로젝트 화면별 적용 문구를 관리합니다.

## X2 화면별 상태 문구

### Intelligence 메인

| 상태 | 제목 | 설명 | CTA |
|------|------|------|-----|
| Empty (첫 방문) | 키워드를 입력하면 분석이 시작돼요 | 검색 데이터, 소셜 반응, 업종 기준을 함께 분석해요 | (입력창 포커스) |
| Loading | — | 분석하고 있어요 | — |
| Error | 분석을 마치지 못했어요 | 잠시 후 다시 시도하면 결과를 볼 수 있어요 | 다시 분석하기 |
| Partial | 일부 데이터 | 연결된 데이터 범위가 아직 넓지 않아요 | — |
| Stale | 이전 데이터 | 최신이 아닐 수 있어요 | — |
| Low confidence | 참고용 | 신뢰도가 낮아요. 데이터가 더 쌓이면 정확도가 올라가요 | — |
| No insights | — | 아직 인사이트가 없어요 | — |

### Intelligence 이력

| 상태 | 제목 | 설명 | CTA |
|------|------|------|-----|
| Empty | 분석 이력이 아직 없어요 | 키워드를 분석하면 이력이 여기에 쌓여요 | 첫 분석 시작하기 |
| Loading | — | (skeleton) | — |

### Intelligence 비교 (변화 탭)

| 상태 | 제목 | 설명 | CTA |
|------|------|------|-----|
| No previous | 비교할 이전 결과가 아직 없어요 | 같은 키워드를 한 번 더 분석하면 바로 비교할 수 있어요 | 다시 분석하기 |
| Insufficient | 이전 데이터를 불러오지 못했어요 | 잠시 후 다시 시도하면 볼 수 있어요 | 다시 불러오기 |

### Compare 화면

| 상태 | 제목 | 설명 | CTA |
|------|------|------|-----|
| Period insufficient (양쪽) | 기간 비교 데이터 부족 | 양쪽 다 저장된 과거 데이터가 없어요. 분석을 먼저 해주세요. | — |
| Period insufficient (한쪽) | 기간 비교 데이터 부족 | 한쪽의 과거 데이터가 없어서 현재 분석으로 대체했어요 | — |
| No differences | — | 눈에 띄는 차이가 없어요. 거의 비슷한 결과예요. | — |

### Bell 알림

| 상태 | 제목 | 설명 | CTA |
|------|------|------|-----|

| Error | — | 알림을 불러오지 못했어요 | 다시 시도 |
| Loading | — | 알림을 불러오는 중이에요 | — |
