# Sometrend Gap Analysis

> Date: 2026-03-16

## 핵심 Gap (썸트렌드 대비 부족)

### Gap 1: 키워드 추이 시계열 세분화
- **썸트렌드:** 일별/주별/월별/채널별 추이 차트
- **x2:** 일별 benchmarkTrend만 있음. 주별/월별 집계 없음. 채널별 시계열 없음.
- **영향:** 중장기 트렌드 분석 약함
- **우선순위:** P1

### Gap 2: 연관어 분석
- **썸트렌드:** 연관어 변화, 연관어 맵, TPO/브랜드/상품 연관
- **x2:** taxonomyMapping(카테고리 매핑)만 있음. 연관어 변화/맵 없음.
- **영향:** 키워드 맥락 이해 약함
- **우선순위:** P1

### Gap 3: 시간별 실시간 추이
- **썸트렌드:** 시간대별 추이, 실시간 모니터링 대시보드
- **x2:** 일별 snapshot만. 시간별 granularity 없음.
- **영향:** 실시간 이슈 대응 약함
- **우선순위:** P2

### Gap 4: 채널 랭킹 / YouTube 전용 분석
- **썸트렌드:** 채널 랭킹, 영상별 성과, 유튜버 분석
- **x2:** DB 스키마 있음(Channel/Content/Metric), YouTube API 연결됨. 전용 대시보드 UI 없음.
- **영향:** YouTube 분석 depth 부족
- **우선순위:** P2

### Gap 5: 데이터 다운로드
- **썸트렌드:** 차트 이미지 다운로드, 엑셀 데이터 다운로드
- **x2:** Export 서비스 구조 있음(Word/PPT/PDF). 차트 이미지/엑셀 직접 다운로드 미구현.
- **영향:** 보고서 작성 효율 저하
- **우선순위:** P2

### Gap 6: 감성 연관어 / 속성별 해석
- **썸트렌드:** 감성과 연관된 키워드, 속성별 강점/약점
- **x2:** 감성 분류는 완료. 감성+키워드 교차 분석 없음.
- **우선순위:** P2
