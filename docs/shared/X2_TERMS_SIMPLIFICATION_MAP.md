# X2 Terms Simplification Map

> X2 프로젝트에서 사용되는 기술 용어를 사용자 친화적 표현으로 치환한 전체 매핑표

## 핵심 개념

| 내부/기술 용어 | 사용자 표현 | 적용 화면 |
|---------------|------------|----------|
| Intelligence Hub | 인사이트 허브 | 헤더 |
| Signal Fusion | 신호 종합 | 탭, 기능 카드 |
| Taxonomy / Taxonomy Mapping | 주제 분류 | 탭 |
| Benchmark Baseline | 업종 기준선 | 카드, 비교 |
| Benchmark Differential | 기준 대비 차이 | 비교 결과 |
| Provider Coverage | 연결된 데이터 범위 | 뱃지, 패널 |
| Live Mentions / 소셜 멘션 | 실시간 반응 | 탭, 패널 |
| Buzz Level | 반응 활동 | 통계 카드 |
| Social Mention Snapshot | 소셜 반응 기록 | 내부 |
| Analysis Run | 분석 결과 | 이력 |
| Comparison Run | 비교 결과 | 비교 |
