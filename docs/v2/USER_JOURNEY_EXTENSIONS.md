## 1. 확장된 페르소나

### P1: 브랜드 마케터 "지은" (썸트렌드 + 리스닝마인드 DNA)

```
역할:    화장품 브랜드 마케팅 팀장
목표:    월간 소셜 성과 보고서를 빠르게 만들고, 다음 달 콘텐츠 방향 결정
Pain:    매주 5시간 이상 수동으로 데이터 수집 + PPT 작성
핵심 여정:
  ② Analyze: 자사 채널 성과 확인, 차트 캡처
  ③ Intent: "선크림 추천" 검색 트렌드 확인
  ⑤ Insight: 주간 보고서 1클릭 생성 → PPT 다운로드
  ④ GEO: "AI가 우리 제품을 추천하는가?" 확인
```

### P2: 에이전시 AE "현수" (소머즈 + 피처링 DNA)

```
역할:    디지털 마케팅 에이전시 AE, 클라이언트 5개 관리
목표:    인플루언서 캠페인 기획 → 실행 → 성과 보고
Pain:    인플루언서 탐색에 2일, 성과 수집에 1일 소요
핵심 여정:
  ① Discover: 뷰티 카테고리 인플루언서 검색 + 필터
  ⑥ Execute: 캠페인 생성, 인플루언서 3명 섭외
  ⑦ Measure: 캠페인 종료 후 성과 자동 수집 → ROI 리포트
  ② Analyze: 클라이언트 채널 원시 데이터 CSV 내보내기
```

### P3: 1인 크리에이터 "민지" (리스닝마인드 + 기본 분석)

```
역할:    유튜브 뷰티 크리에이터 (구독자 5만)
목표:    다음 영상 주제 결정, 댓글 관리
Pain:    "뭘 찍어야 조회수가 나올지 모르겠다"
핵심 여정:
  ③ Intent: "여드름 관리" 검색 의도 탐색 → 블루오션 키워드 발견
  ② Analyze: 최근 영상 성과 확인 + 댓글 감성 추이
  ⑤ Insight: "이 키워드로 영상을 만들면 조회수 예상 XX" 인사이트
```

### P4: Enterprise 마케팅 디렉터 "성호" (아하트렌드 + 전체)

```
역할:    대기업 마케팅 본부장, 팀 15명
목표:    전사 소셜 전략 수립 + 경쟁사 대비 포지셔닝
Pain:    여러 부서가 다른 도구 사용, 데이터 통합 불가
핵심 여정:
  전체 7스테이지 + Vertical Pack(F&B) + Enterprise SSO
  핵심: ⑦ Measure 벤치마크로 산업 평균 대비 자사 위치 파악
```

## 3. 스테이지 간 전환 포인트

사용자가 자연스럽게 다음 스테이지로 이동하는 CTA:

| 현재 → 다음 | 전환 CTA                                       | 예시                     |
| ----------- | ---------------------------------------------- | ------------------------ |
| ① → ②       | "이 채널 상세 분석 보기"                       | 인플루언서 카드 클릭     |
| ② → ③       | "이 콘텐츠 관련 검색 트렌드 보기"              | 인기 콘텐츠 옆 링크      |
| ③ → ④       | "이 키워드가 AI 검색에서 어떻게 나오는지 확인" | 갭 분석 결과 옆          |
| ④ → ⑤       | "인사이트 요약 보기"                           | 가시성 변화 알림         |
| ⑤ → ⑥       | "이 제안을 실행하기"                           | 인사이트 카드 액션 버튼  |
| ⑥ → ⑦       | "캠페인 성과 확인"                             | 캠페인 종료 시 자동 알림 |
| ⑦ → ①       | "다음 캠페인을 위한 인플루언서 탐색"           | 성과 리포트 하단 CTA     |

## 4. Vertical Pack이 여정에 미치는 영향

Vertical Pack 활성화 시 각 스테이지에 추가되는 요소:

| 스테이지   | 범용 (기본)        | + Vertical Pack (예: 뷰티)               |
| ---------- | ------------------ | ---------------------------------------- |
| ① Discover | 전체 카테고리 검색 | 뷰티 카테고리 프리셋, 성분 필터          |
| ② Analyze  | 일반 지표          | 뷰티 특화 토픽 분류 (성분, 텍스처, 효과) |
| ③ Intent   | 일반 의도 분류     | 뷰티 키워드 사전 500+ 적용               |
| ④ GEO      | 일반 Citation 추적 | 뷰티 키워드 프리셋 자동 등록             |
| ⑤ Insight  | 일반 보고서        | 뷰티 리포트 템플릿, 업계 벤치마크 포함   |
| ⑥ Execute  | 일반 캠페인        | 뷰티 인플루언서 자동 추천                |
| ⑦ Measure  | 일반 성과          | 뷰티 업계 평균 대비 ROI 비교             |
