## 1. 유저 여정 (End-to-End)

### Phase A: 발견 & 체험 (비로그인)

```
랜딩 페이지 도착
  ├─ [채널 분석 체험] URL 입력 → 부분 결과 (잠금)
  ├─ [인텐트 분석 체험] 키워드 입력 → 부분 결과 (잠금)
  └─ CTA: "전체 결과를 보려면 가입하세요"
       ↓
  회원가입 (Google / Naver / Kakao)
```

### Phase B: 온보딩 (가입 직후)

```
온보딩 위저드 (3단계)
  ① 워크스페이스 이름 설정
  ② 첫 번째 채널 URL 등록
       → 자동 플랫폼 감지 → 데이터 수집 시작
  ③ 관심 키워드 1~3개 입력
       → Search Intent 분석 큐 등록
       ↓
  대시보드 (수집 진행 상태 표시)
```

### Phase C: 일상 사용 루프

```
[매일]
  대시보드 확인
    → KPI 변동 알림 (구독자↑, 참여율↓ 등)
    → 새 댓글 요약 (감성 변화, 리스크 감지)
    → AI 인사이트 카드 확인

[매주]
  Search Intent 탐색
    → 트렌드 키워드 발견
    → 소셜 갭 분석 (검색 수요 ↔ 콘텐츠 공급 비교)
    → 콘텐츠 아이디어 도출

  GEO/AEO 체크
    → "우리 브랜드가 AI 답변에 나오는가?"
    → 경쟁사 AI 가시성 비교

  Comment Intelligence 리뷰
    → 토픽별 댓글 클러스터
    → FAQ 자동 추출
    → 부정 댓글 알림 + 대응 제안

[매월]
  리포트 생성
    → 월간 성과 요약 (자동 생성)
    → 경쟁 벤치마크
    → 전략 제안 액션 아이템

  Action Automation 리뷰
    → 완료된 액션 성과 확인
    → 새 액션 아이템 승인/수정
```

### Phase D: 심화 사용

```
경쟁 분석 심화
  → 경쟁 채널 추가 (최대 5개/Pro)
  → 채널 비교 매트릭스
  → 경쟁사 콘텐츠 전략 분석

워크스페이스 확장
  → 팀 멤버 초대
  → 역할 기반 접근 제어
  → 클라이언트 워크스페이스 분리

API 연동
  → Business: REST API 제공
  → 커스텀 대시보드 구축
  → Slack/Teams 웹훅 연동
```

## 2. 정보 아키텍처 (IA)

### 네비게이션 구조 (v2)

```
X2 Platform
│
├── 🏠 대시보드                          ← 통합 개요 + KPI + 알림
│
├── 🔍 Search Intent                    ← [모듈 1]
│   ├── 키워드 탐색                       ← 새 키워드 분석 시작
│   ├── 인텐트 맵                         ← intent graph 시각화
│   ├── 갭 분석                           ← 검색 수요 vs 콘텐츠 공급
│   └── 트렌드                            ← 시계열 트렌드 + 시즈널
│
├── 📊 Social Intelligence              ← [모듈 2]
│   ├── 채널 목록                         ← 등록 채널 관리
│   ├── 채널 상세                         ← 개별 채널 분석
│   ├── 콘텐츠 라이브러리                  ← 전체 콘텐츠 목록/필터
│   ├── 성과 분석                         ← 채널별/기간별 성과 차트
│   └── 경쟁 비교                         ← 채널 vs 채널 비교
│
├── 💬 Comment Intelligence             ← [모듈 3]
│   ├── 댓글 대시보드                      ← 감성 트렌드 + 요약
│   ├── 토픽 클러스터                      ← 주제별 댓글 그룹
│   ├── FAQ 추출                          ← 자주 묻는 질문 자동 감지
│   ├── 리스크 알림                        ← 부정/위기 감지
│   └── 댓글 목록                          ← 전체 댓글 검색/필터
│
├── 🤖 GEO / AEO                        ← [모듈 4]
│   ├── AI 가시성 추적                     ← AI 검색엔진 결과 모니터링
│   ├── 키워드별 AI 답변                   ← 어떤 소스가 인용되는가
│   ├── 경쟁 가시성 비교                   ← 우리 vs 경쟁사 AI 노출
│   └── 최적화 제안                        ← GEO/AEO 전략 가이드
│
├── ⚡ Action Center                     ← [모듈 5]
│   ├── 액션 아이템                        ← AI 생성 + 수동 태스크
│   ├── 콘텐츠 캘린더                      ← 발행 스케줄 + 제안
│   ├── 자동화 규칙                        ← 조건부 자동 실행
│   └── 실행 히스토리                      ← 완료된 액션 + 성과
│
├── 📄 리포트                             ← 크로스 모듈
│   ├── 자동 리포트                        ← 주간/월간 자동 생성
│   ├── 커스텀 리포트                      ← 템플릿 기반 수동 생성
│   └── 공유 리포트                        ← 외부 공유 링크
│
├── ⚙️ 설정
│   ├── 워크스페이스                       ← 이름, 멤버, 역할
│   ├── 연동 관리                          ← OAuth, API 키
│   ├── 알림 설정                          ← 이메일, 인앱, Slack
│   ├── 요금제 & 결제                      ← 플랜 변경, 결제 내역
│   └── 프로필                             ← 개인 정보, 언어
│
└── 🛡️ 관리자 (Admin)
    ├── 사용자 관리
    ├── 플랫폼 상태
    ├── 수집 관제
    ├── AI 사용량/비용
    └── 시스템 로그
```

## 3. 화면별 데이터 요구사항

### 대시보드

| 섹션           | 데이터 소스              | 갱신 주기    |
| -------------- | ------------------------ | ------------ |
| KPI 카드 (6개) | ChannelSnapshot 집계     | 일 1회       |
| 성과 차트      | ChannelSnapshot 시계열   | 일 1회       |
| 최근 인사이트  | InsightReport (최신 3건) | AI 생성 시   |
| 트렌드 키워드  | KeywordMetricDaily       | 일 1회       |
| 댓글 요약      | CommentAnalysis 집계     | 실시간~1시간 |
| 액션 아이템    | InsightAction (미완료)   | 생성 시      |

### Search Intent — 키워드 탐색

| 섹션          | 데이터 소스                    | 갱신 주기 |
| ------------- | ------------------------------ | --------- |
| 키워드 입력   | 사용자 입력                    | 온디맨드  |
| 인텐트 분류   | IntentEngine → LLM             | 분석 시   |
| 검색량 트렌드 | Google Trends API / DataForSEO | 분석 시   |
| 소셜 볼륨     | Social API 검색                | 분석 시   |
| 갭 점수       | 검색량 vs 소셜 콘텐츠 비율     | 분석 시   |
| 인텐트 그래프 | IntentEngine graph builder     | 분석 시   |

### Social Intelligence — 채널 상세

| 섹션        | 데이터 소스                  | 갱신 주기        |
| ----------- | ---------------------------- | ---------------- |
| 채널 프로필 | Channel + Platform API       | 등록 시 + 일 1회 |
| 지표 카드   | ChannelSnapshot (최신)       | 일 1회           |
| 추이 차트   | ChannelSnapshot 시계열       | 일 1회           |
| 콘텐츠 목록 | Content + ContentMetricDaily | 일 1회           |
| 업로드 빈도 | Content publishedAt 집계     | 일 1회           |
| AI 인사이트 | InsightReport (채널별)       | 주 1회           |

### Comment Intelligence — 대시보드

| 섹션          | 데이터 소스                              | 갱신 주기 |
| ------------- | ---------------------------------------- | --------- |
| 감성 분포     | CommentAnalysis 집계                     | 수집 시   |
| 감성 트렌드   | CommentAnalysis 일별 집계                | 일 1회    |
| 토픽 클러스터 | CommentAnalysis topics                   | 수집 시   |
| FAQ 추출      | AI 분석 (Comment batch)                  | 주 1회    |
| 리스크 알림   | CommentAnalysis (negative + high volume) | 실시간    |
| 워드클라우드  | Comment text 형태소 분석                 | 주 1회    |

### GEO/AEO — AI 가시성

| 섹션             | 데이터 소스                    | 갱신 주기 |

## 4. 상태 패턴 (Empty / Loading / Error / Data)

모든 화면은 4가지 상태를 명시적으로 처리한다:

### Empty State

- **채널 없음**: "첫 번째 채널을 등록하세요" + URL 입력 폼
- **데이터 수집 중**: "데이터를 수집하고 있습니다 (약 2분)" + 진행 바
- **분석 없음**: "키워드를 입력하면 인텐트 분석이 시작됩니다"

### Loading State

- 스켈레톤 UI (실제 레이아웃과 동일한 형태)
- 수집/분석 중: 단계별 진행 상태 표시

### Error State

- API 키 미설정: "YouTube API 키를 설정하면 실데이터를 수집할 수 있습니다" + 설정 링크
- 수집 실패: "데이터 수집에 실패했습니다. [재시도]"
- 할당량 초과: "이번 달 분석 할당량을 초과했습니다. [업그레이드]"

### Data State

- 실데이터 표시
- 마지막 갱신 시각 표시
- 데이터 신선도 인디케이터 (초록: 24h 이내, 노랑: 7일 이내, 빨강: 7일 초과)
