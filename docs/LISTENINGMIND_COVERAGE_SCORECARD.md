# ListeningMind Coverage Scorecard

## 평가 기준
- **A. 구현+검증 완료**: 실제 코드 + 실데이터 연결 + usable 상태
- **B. 부분 구현**: UI/엔진 존재하나 데이터 미연결, mock 기반, 또는 검증 미완
- **C. 미구현**: 실질적 구현 없음
- **D. 불명확**: 흔적 있으나 확인 필요

---

## 1. Intent Finder 축

| 체크포인트 | 상태 | 근거 |
|-----------|------|------|
| 단일 키워드 심층 탐색 | **A** | intent/page.tsx (742줄), IntentClassifier 40+ 패턴 |
| 다중 키워드 입력 | **B** | 단일 seed → 자동 확장 (15개), 직접 다중 입력 UI 없음 |
| 관련 검색 표현 | **A** | relatedKeywords, autocomplete 어댑터 연동 |
| 주요 토픽/관심사 도출 | **A** | 토픽 추출 + 감성 연관어 + 갭 매트릭스 |
| **축 종합** | **A (85%)** | 엔진 완성, 다중 키워드 직접 입력만 부족 |

|-----------|------|------|
