# Roadmap

## 개발 단계

### Phase 0: 환경 세팅 (1주)

- [x] Git 저장소 초기화
- [ ] Turborepo 모노레포 구성
- [ ] TypeScript, ESLint, Prettier 설정
- [ ] TailwindCSS + shadcn/ui 설정
- [ ] Prisma 스키마 초기 설계 + DB 연결 (Supabase)
- [ ] Auth.js 소셜 로그인 (Google)
- [ ] tRPC 기본 구조 세팅
- [ ] CI/CD: Vercel 배포 파이프라인
- [ ] 환경변수 관리 (.env.example)

### Phase 1: MVP 핵심 기능 (4~5주)

#### 1-1. 채널 관리 (1주)

- [ ] 채널 URL 입력으로 플랫폼 자동 감지
- [ ] YouTube 채널 기본 정보 수집 (공개 데이터)
- [ ] 채널 목록 CRUD (추가, 수정, 삭제)
- [ ] 채널 상세 페이지

#### 1-2. 기본 분석 대시보드 (1.5주)

- [ ] 대시보드 레이아웃 (사이드바, 헤더, 메인)
- [ ] 채널 개요 카드 (구독자, 영상 수, 총 조회수)
- [ ] 최근 콘텐츠 목록 및 성과 지표
- [ ] 기간별 추이 차트 (Recharts)
- [ ] 간단한 경쟁 채널 비교

#### 1-3. 콘텐츠 분석 (1주)

- [ ] YouTube 영상 목록 수집 및 표시
- [ ] 영상별 조회수, 좋아요, 댓글 수 추이
- [ ] 상위 성과 콘텐츠 하이라이트

#### 1-4. AI 인사이트 (1주)

- [ ] 채널 성과 요약 (Claude API)
- [ ] 콘텐츠 개선 제안
- [ ] 간단한 댓글 감성 분석

#### 1-5. 사용자 및 과금 (0.5주)

- [ ] 사용자 프로필/설정 페이지
- [ ] Free/Pro 요금제 정의
- [ ] Stripe Checkout 연동 (Pro 결제)
- [ ] Feature Gate 미들웨어

### Phase 2: 확장 (4~6주)

- [ ] Instagram, TikTok, X 플랫폼 추가
- [ ] 공식 API 연동 (OAuth) → 상세 분석 데이터
- [ ] 키워드 검색 및 소셜 리스닝
- [ ] 리포트 생성 및 PDF 내보내기
- [ ] 댓글 상세 분석 (토픽 분류, 감성 점수)
- [ ] 스케줄링 (자동 데이터 수집)
- [ ] 트렌드 분석 기본

### Phase 3: 고도화 (4~6주)

- [ ] 트렌드 예측 모델
- [ ] 단기/중기/장기 전략 제안 (AI)
- [ ] 경쟁 채널 심층 분석
- [ ] 알림 시스템 (이메일, 인앱)
- [ ] Python 워커 도입 (데이터 파이프라인)
- [ ] Business 요금제 + 팀 기능
- [ ] TossPayments 국내 결제 추가

### Phase 4: 모바일 앱

- [ ] React Native (Expo) 앱 개발
- [ ] 핵심 대시보드 모바일 UI
- [ ] 푸시 알림
- [ ] App Store / Google Play 출시

## MVP 범위 정의

### MVP에 반드시 포함 (Must Have)

| 기능        | 설명                                        |
| ----------- | ------------------------------------------- |
| 소셜 로그인 | Google OAuth                                |
| 채널 등록   | URL 입력으로 YouTube 채널 추가              |
| 기본 분석   | 구독자, 조회수, 영상 수 등 공개 데이터 기반 |
| 대시보드    | 채널 개요, 추이 차트, 콘텐츠 목록           |
| 콘텐츠 목록 | 영상별 성과 지표                            |
| AI 요약     | 채널 성과 요약 및 개선 제안                 |
| 과금        | Free/Pro 2-tier, Stripe 결제                |
| 반응형 웹   | 모바일 웹 대응                              |

### 후순위 (Deferred)

| 기능                  | 이유                                     |
| --------------------- | ---------------------------------------- |
| Instagram/TikTok/X    | MVP는 YouTube 단일 채널로 검증           |
| 공식 API 연동 (OAuth) | 기본 분석으로 먼저 가치 검증             |
| 키워드 리스닝         | 별도 데이터 파이프라인 필요, 복잡도 높음 |
| 리포트 PDF 생성       | 대시보드로 먼저 가치 전달                |
| 댓글 상세 분석        | AI 비용 최적화 후 도입                   |
| 트렌드 예측           | ML 모델 필요, Phase 3                    |
| 스케줄링/자동 분석    | 수동 분석으로 시작                       |
| 경쟁 채널 심층 분석   | 기본 비교는 포함, 심층은 후순위          |
| 팀 기능/멀티 시트     | B2B 확장 시 도입                         |
| 모바일 앱             | 웹 MVP 검증 후                           |
| Kakao/Naver 로그인    | 국내 런칭 확정 시                        |
