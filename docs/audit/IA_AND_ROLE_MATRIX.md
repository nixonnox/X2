## 1. 현재 메뉴 구조

### 사이드바 네비게이션 (현재)

```
개요
├── 대시보드

분석
├── 채널 관리
├── 채널 성과
├── 콘텐츠
├── 댓글 분석
├── 경쟁 분석
├── 키워드/트렌드

인사이트
├── AI 인사이트
├── 추천 액션
├── 리포트

계정
├── 설정
├── 요금제

관리 (접힘 가능)
├── AI 분석 관리
├── AI 프롬프트
├── AI 로그
├── AI 평가
├── 데이터 수집
├── 사용자 관리
├── 플랫폼 관리
├── 요금제 관리
├── 시스템 로그
```

### 현재 문제점

1. **관리 메뉴가 모든 사용자에게 보임** — 권한 체크 없음
2. **채널 관리 vs 채널 성과 구분 불명확** — 사용자 혼란
3. **콘텐츠가 독립 메뉴** — 채널 하위가 더 자연스러움
4. **AI 관리 하위 메뉴가 4개** — 관리자에게만 필요
5. **키워드/트렌드와 멘션이 분리** — 리스닝 통합 필요
6. **빌링이 "계정" 아래** — 관리자 성격에 더 가까움

## 2. 개선된 메뉴 구조 (제안)

### 일반 사용자 메뉴

```
홈
├── 대시보드                    ← 내 채널 종합 현황

채널
├── 내 채널                     ← 등록된 채널 목록 + 등록
├── 채널 분석                   ← 개별 채널 상세 분석
├── 콘텐츠 성과                 ← 전체 콘텐츠 통합 뷰

댓글
├── 댓글 분석                   ← 감성/리스크/FAQ
├── 댓글 관리                   ← 답변 관리

경쟁 분석
├── 경쟁 채널                   ← 등록 + 비교
├── 벤치마크                    ← 업계 비교

리스닝
├── 키워드 모니터링              ← 등록 + 트렌드
├── 멘션 분석                   ← 멘션 수집 + 분석

인사이트
├── AI 인사이트                 ← 자동 생성 인사이트
├── 추천 액션                   ← 실행 가능한 제안
├── 리포트                      ← 생성 + 스케줄 + 공유

설정
├── 계정 설정                   ← 프로필, 비밀번호
├── API 키 관리                 ← 플랫폼 API 키 설정 ★신규
├── 요금제 & 결제               ← 현재 플랜, 업그레이드
├── 알림 설정                   ← 이메일/푸시 설정 ★신규
```

### 관리자 전용 메뉴 (OWNER/ADMIN만 표시)

```
관리자
├── 워크스페이스 관리            ← 멤버 초대/역할
├── 데이터 수집 관리            ← 커넥터 상태, 작업 이력
├── AI 운영 관리                ← 프로바이더 상태, 비용, 프롬프트
├── 사용량 & 한도               ← 워크스페이스 사용량
├── 시스템 상태                 ← 헬스, 로그
```

## 3. 역할 정의

| 역할                | 코드         | 설명                       |
| ------------------- | ------------ | -------------------------- |
| 비회원              | `GUEST`      | 가입 전 방문자             |
| 무료 회원           | `FREE`       | 기본 기능만 사용 가능      |

| Enterprise          | `ENTERPRISE` | 무제한, 커스텀             |

| 워크스페이스 멤버   | `MEMBER`     | 분석 기능만                |
